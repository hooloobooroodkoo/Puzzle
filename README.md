# Puzzle

This module is created for a game "Puzzle". With the help of this module you can check game board to know whether board is ready for game, whether all placements of numbers correspond to the rules of game.

This module consists of following functions:

def check_rows(board) - returns True if rows consists only of free spaces, white spaces and number 1-9

def columns(board) - returns list of columns on the board

def check_columns(board) - returns True if columns consists only of free spaces, white spaces and number 1-9

def create_color_blocks(board) - creates list of spaces painted in one color on the board

def check_color_blocks(board) - returns True if columns consists only of free spaces, white spaces and number 1-9

def validate_board(board) - checks whether board is correct
